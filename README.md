# Personal Site (Instagram-style)

Статичный личный сайт-профиль в стиле Instagram: шапка профиля, лента историй, лента публикаций 3×3 и модальное окно просмотра поста.

## Запуск

Открыть `index.html` в браузере, либо поднять локальный сервер:

```bash
cd personal-site
python3 -m http.server 8000
```

и перейти на `http://localhost:8000`.

## Структура

```
personal-site/
├── index.html              # разметка страницы
├── assets/
│   ├── css/style.css       # стили в духе Instagram
│   ├── js/script.js        # данные постов/историй и логика (модалка, табы)
│   └── images/
│       ├── posts/          # сюда положить свои фото постов
│       └── stories/        # сюда положить аватарки для историй
```

## Как наполнить своим контентом

1. **Профиль**: имя пользователя, био и ссылка редактируются прямо в `index.html` (секция `.profile-header`).
2. **Публикации**: отредактировать массив `POSTS` в `assets/js/script.js`. Если положить фото в `assets/images/posts/` и указать путь в поле `image`, вместо цветной заглушки покажется реальное фото.
3. **Истории**: массив `STORIES` в том же файле — задать подписи и эмодзи/картинки.

Сайт полностью статический — HTML/CSS/JS без сборки, можно разместить на GitHub Pages, Netlify или любом хостинге.
